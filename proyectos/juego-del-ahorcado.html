Contexto

El ahorcado es un juego de palabras sencillo pero perfecto para practicar lógica de programación: adivinar una palabra letra a letra antes de agotar los intentos. Lo llevé a la web con JavaScript puro.

El problema

El reto era modelar correctamente el estado del juego —palabra oculta, letras acertadas, intentos restantes— y reflejarlo en una interfaz que respondiera de inmediato a cada jugada.

Mi rol

Desarrollo frontend completo del juego.

  • Diseño de la lógica de estado del juego.
  • Manipulación del DOM para reflejar cada jugada.
  • Interfaz y estilos con HTML5 y CSS3.

La solución

Implementé el juego con JavaScript vanilla, gestionando el estado en memoria y actualizando la interfaz de forma reactiva ante cada letra elegida. Sin librerías: solo fundamentos.

  • Selección de palabra y control de intentos.
  • Detección de victoria y derrota.
  • Interfaz interactiva con retroalimentación visual.
