Gestor de Tienda de Videojuegos

Aplicación de gestión de inventario, ventas y catálogo para una tienda especializada en videojuegos.

C# .NET

Captura principal del proyecto

Contexto

Una tienda de videojuegos necesita mantener bajo control su inventario, registrar las ventas y organizar un catálogo que cambia constantemente. Este proyecto centraliza toda esa gestión en una sola aplicación.

El problema

Llevar el inventario y las ventas de forma manual es propenso a errores y difícil de escalar. El objetivo era ofrecer una herramienta fiable que centralizara productos, existencias y operaciones de venta.

Mi rol

Desarrollo completo de la aplicación, de la lógica de negocio a la interfaz.

  • Diseño del modelo de datos (productos, stock, ventas).
  • Implementación de las operaciones CRUD y el registro de ventas.
  • Construcción de la interfaz de usuario con C# y .NET.

La solución

Desarrollé la aplicación con C# sobre .NET, separando la lógica de negocio de la capa de datos para mantener el código ordenado y fácil de ampliar.

  • Gestión de catálogo con altas, bajas y modificaciones.
  • Control de existencias en tiempo real.
  • Registro de ventas con actualización automática del inventario.
