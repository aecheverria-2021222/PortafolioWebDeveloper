Sistema de Gestión Veterinaria

Aplicación para gestionar historiales clínicos, citas y pacientes de una clínica veterinaria.

Captura principal del proyecto

Contexto

Una clínica veterinaria maneja mucha información sensible: datos de los dueños, fichas de las mascotas, historiales clínicos y una agenda de citas. Este sistema digitaliza y centraliza toda esa gestión.

El problema

Sin una herramienta adecuada, la información clínica se dispersa y se vuelve difícil de consultar. El reto era ofrecer una aplicación clara para registrar pacientes, agendar citas y acceder rápidamente a los historiales.

Mi rol

Desarrollo completo de la aplicación de escritorio y su base de datos.

  • Modelado de entidades: dueños, mascotas, citas e historiales.
  • Interfaz de escritorio con JavaFX.
  • Persistencia y consultas sobre MySQL.

La solución

Construí la aplicación con Java y una interfaz en JavaFX, con los datos almacenados en MySQL. El diseño prioriza el acceso rápido a la información que un veterinario necesita durante una consulta.

  • Gestión de pacientes y sus dueños.
  • Agenda de citas.
  • Historiales clínicos consultables por mascota.
